cross roads c.a.c.

A redesign project of the landing and about us page for Cross Roads C.A.C. The main goal of this project is to create a more tidy and user-friendly interface, with stylistic edits and a more effective use of graphics and images.

Cross Roads C.A.C. is a non-profit organization that provides assistance and services for affordable housing within low-income communities.
